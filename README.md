# (Next.js + Tailwindcss V4 + TypeScript) responsive boilerplate

- navbar with hamburger menu
- main with mid/large sections container
- footer with go-to-top button

## How to start:

Install the dependencies

```bash
npm i
```

Run the local server

```bash
npm run dev
```


## Usefull commands:
-npm run dev
-npm run build
-npm start

-npm outdated
